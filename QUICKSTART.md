# Quick Start Guide - Murder Mystery Game with Gemini AI

## 🚀 Get Started in 3 Steps

### Step 1: Install Package
```bash
pip install google-generativeai
```

### Step 2: Get Free API Key
Visit: **https://makersuite.google.com/app/apikey** and create an API key

### Step 3: Run Game
```bash
python main.py
```

When prompted, paste your API key and watch the game generate a unique mystery!

---

## What's New? 🎮

Every time you play, Gemini AI generates:

✨ **Unique Victims** - Different names, ages, professions
✨ **Original Suspects** - New characters with unique motives
✨ **Dynamic Alibis** - Different stories each game
✨ **Smart Clues** - Crime scene evidence specific to the mystery
✨ **Hidden Contradictions** - Conflicts between alibis and clues

---

## How to Play

1. **Interrogate suspects** - Ask them about their alibis
2. **Find contradictions** - Spot conflicts with crime scene clues
3. **Earn points** - Get 10 points per contradiction found
4. **Make accusation** - Accuse the murderer and get 50 bonus points
5. **Repeat** - Each new game has a completely different mystery!

## Troubleshooting

**Game running but not using AI?**
- Check internet connection
- Verify API key is correct
- The game has fallback data, so it'll still work!

**Rate limited?**
- Free tier allows 60 requests/minute
- Wait a few minutes and try again

**Need help?**
- Check `SETUP_GUIDE.md` for detailed instructions
- See error messages in console for specific issues
